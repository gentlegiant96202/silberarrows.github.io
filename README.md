# Silber Arrows Website

This repository contains the source code for the Silber Arrows website, a premium Mercedes-Benz service center in Dubai.

## Live Site

The website is hosted on GitHub Pages and can be viewed at: [https://gentlegiant96202.github.io/silberarrows.github.io/](https://gentlegiant96202.github.io/silberarrows.github.io/)

## Development

To run the website locally, you can use Python's built-in HTTP server:

```bash
python3 -m http.server 8000
```

Then, open your browser and navigate to `http://localhost:8000`.
